Description

Nokia Demo Center System contains different demos developed by Nokia Experience Lab members for Nokia Pre-sales people. They have technical backgrounds and are always looking for ways to demostrate Nokia Software products with a high degree of customization to close sales better.

Functionalities

Feedback System

The feedback system allows the user to rate and give feedback for the each individual demo as well as the system. By collecting the feedback, Nokia can improve the demo system to identify and fulfill users' needs.

Front End

The frontend for feedback system a multi-step form, which guide the user to rate, enter feedback, authenticate the data, and send them to the server via AJAX. To improve the UX, I made a flip card effect to combine feedback system with each individual demo.

Customer View
Admin View

A feedback view is also created in the admin view. The admin, with special settings, will be able to check the average rating and feedbacks for individual demos and the system.

Static Demo Listing

When user navigate to /clickdemo page, lists of static demos will be retrieved from the Postgres database. Django will render the details and link to the demo on the webpage.
