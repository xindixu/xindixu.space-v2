Project Description

Background

During the summer of 2018, I interned at Daikin of North America, Goodman Manufacturing LLC. Our-next-branch is my personal project. This was my first time working with Tableau and I feel so honored until today when I learned that the company was actually using this visualization to make decisions.

Goal

This tool is designed to help users find out which area in North America has an insufficient supply of HVAC distributors.

Data

There are three categories of data: demand, supply, and performance. Users can toggle filters to narrow down to specific data range, as well as highlight a certain value in a specific field. They can also collapse tabs to switch views among state, city and zip code.

Values

Demand

From US Census Bureau

  • Household population [HH#]
  • Percent of households using electricity as a source of home heating/cooling fuel [%Electricity]
  • Percent of the houses built in 2000-2009 (10-20 years old) [%10-20]

Supply

From distributor company websites

  • Company name at the location
  • Category (COD, IND, Competitors’ COD, Distributors not carrying our brand)
  • Brands the location is carrying
  • Address of the location
  • Total number of distributors in an area [D#]

Performance

From Goodman database

  • Goodman market share
  • Goodman unit sales from COD & IND (Jul. 2017 – Jun. 2018)

Geographic Levels

Demand

All: State – City – ZIP Code

Supply

All: State – City – ZIP Code

Performance

Market share: Division – State – Trade Area – County – ZIP Code

Sales: State – Trade Area – County – ZIP Code

1. Scrape Website Data

Since there are more than 60 distributors's website to go through, I turned to ParseHub for web scraping. I first inspected the HTML structure of the webpage and constructed the code to scrape data.

Scraping web data makes future update a lot easier! The company just have to run the scraping script every once a while and collect all data in .csv format.
