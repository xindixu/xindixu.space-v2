Project Description

Background

During the summer of 2018, I interned at Daikin of North America, Goodman Manufacturing LLC. Our-next-branch is my personal project. This was my first time working with Tableau and I feel so honored until today when I learned that the company was actually using this visualization to make decisions.

Goal

This tool is designed to help users find out which area in North America has an insufficient supply of HVAC distributors.

Steps

1. Scrape Website Data

Since there are more than 60 distributors's website to go through, I turned to ParseHub for web scraping. I first inspected the HTML structure of the webpage and constructed the code to scrape data.

Scraping web data makes future update a lot easier! The company just have to run the scraping script every once a while and collect all data in .csv format.

2. Batch Geocoding with Google Map API

Because the address data I collected are not precise enough, I used Google Map API to standardize the address and batch geocode.
